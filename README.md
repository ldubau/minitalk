# Description
The project of minitalk is to create a communication between two servers named server and client,<br>
the client send strings to the server and the server as to print it into the terminal.<br>

# Instructions
Write ```make``` to create both executable files and for each files, <br>run it in separate terminal at first ```./server``` it will print the the PID like this ```PID : 12345```, now run ```./client PID "Your string"``` with the server PID insted of "PID" (12345 in exemple) and whatever you want between double quote.<br>
The programe should display the string in the server terminal.<br>
Make sur to put the right PID number and the right number of argument otherwise it will doesn't work.
